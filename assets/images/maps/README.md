# Maps Assets

This folder contains map images for the restaurant profile screen.

## Required Files

### Map Image (1).png
- **Purpose**: Display the actual Google Maps view of the restaurant location
- **Size**: Recommended 400x300px or larger
- **Format**: PNG with transparency support
- **Content**: Google Maps screenshot showing the restaurant location with streets and landmarks

## Current Image

The image `Map Image (1).png` is currently being used and shows the area around "Cihangir, Havyar Sk. No:34/A, 34000 Beyoğlu/Istanbul, Türkiye" with the restaurant location marked.

## Fallback

If the image is not found, the app will display a placeholder with a map icon and "Tap to open in Google Maps" text.
